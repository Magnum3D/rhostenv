# rhostenv
Repetier Host environment for Raspberry PI and Magnum3D printers

## Prerequisites

  * RPI 3
  * ```sudo apt update```
  * ```sudo apt -y upgrade```
  * ```sudo apt -y install mc monit minicom wget rpi-update git lscpu lshw cpuinfo```
  
## Setup

  * ```git clone https://github.com/Magnum3D/rhostenv/```
